## ğŸ¨ Step-by-Step Creation Guide

### Step 1: Set Up Your Canvas
```
Tool: Figma / Photoshop / Canva
Canvas Size: 1200 x 900 pixels
Resolution: 72 DPI
Color Mode: RGB
```

### Step 2: Add Background
```
Options:
1. Gradient: Dark blue â†’ Purple (#1a1a2e â†’ #16213e)
2. Solid: Navy (#0F172A) or Slate (#1E293B)
3. Pattern: Subtle dots, grid, or geometric shapes
4. Multiple layers with blur effects
```

### Step 3: Import Device Mockup
```
Resources:
- Figma Community (search "iPhone mockup" or "MacBook mockup")
- Mockuper.net
- Smartmockups.com
- Free mockups from Mockup World

Placement: Center or slightly offset
Size: 60-70% of canvas height for phones, 70-80% for laptops
```

### Step 4: Add Your Screenshot
```
1. Take clean screenshot of your app/website
2. Place inside device mockup
3. Ensure it fits the screen perfectly
4. Check readability of text and UI elements
```

### Step 5: Add Effects
```
Effects to add:
- Drop shadow (Y: 20px, Blur: 40px, Opacity: 40%)
- Glow effect (optional, use brand color)
- Depth with layering
- Subtle decorative elements (circles, lines)
```

### Step 6: Export
```
Format: PNG (for transparency) or JPEG (smaller size)
Quality: High (90-100%)
File size target: < 800KB

Optimize with:
- TinyPNG.com
- ImageOptim (Mac)
- Squoosh.app
```
