## ğŸ–¼ï¸ What to Include in the Image

Your showcase image should be a **complete composition** containing:

1. **Background Layer** (full 1200x900 canvas)
   - Professional gradient, pattern, or solid color
   - Dark colors work best (navy, slate, dark blue)
   - Example: Gradient from #1a1a2e to #16213e

2. **Device Mockup** (centered in the background)
   - Phone mockup for mobile apps
   - Laptop/browser mockup for web apps
   - 60-70% of the canvas height

3. **Your Project Screenshot** (inside the mockup)
   - Real app/website interface
   - Clear and readable
   - No sensitive data

4. **Visual Effects** (for polish)
   - Shadow underneath device
   - Optional glow effects
   - Depth and layering

## ğŸ› ï¸ How to Create

### Option 1: Using Figma (Recommended)

1. Create new file with **1200 x 900px** frame
2. Add background (gradient or solid)
3. Search Figma Community for "iPhone mockup" or "MacBook mockup"
4. Place mockup in center
5. Add your screenshot inside mockup
6. Add shadow effect (Y: 20px, Blur: 40px)
7. Export as PNG

### Option 2: Using Canva

1. Custom size: **1200 x 900 pixels**
2. Choose dark background template
3. Upload device mockup image
4. Add your screenshot
5. Apply effects
6. Download as PNG

### Option 3: Using Photoshop/Photopea

1. New document: **1200 x 900px**, 72 DPI
2. Create gradient background layer
3. Import device mockup
4. Place screenshot as smart object
5. Add layer effects (drop shadow)
6. Export as PNG
